Linkage Plugin Readme

简介

这是一个使用jQuery和select2的联动插件。

使用插件

使用插件前需要先检查:

  1. 它需要jQuery;点击下载jQuery
  2. 它需要select2;点击下载select2

API

var linkage = $(document).Linkage({
"selectors" : ["#s1", "#s2", "#s3"],
"getLinkList" : function(context, selector, param){
// return 你的option数据
}
});
查看例子

Options:
$.fn.Linkage.defaults = {
"nvlOpt" : "请选择",// 这是值为空的option显示的文字，你可以修改成其他的文字 "selectors" : [], // 需要联动的下拉框选择器
"values" : [], // 下拉框的默认值，需要和选择器顺序一致
"getOption" : Linkage.prototype.getOption, // 制作options的方法，可以覆盖自定义实现
"getLinkList" : Linkage.prototype.getLinkList, // 返回联动的数据方法，必须自定义实现
}

option shape:
JSON = {"code":"xx","name":"xx"...} // option的数据为JSON格式，其中code和name属性是必填项，如果还有其他的属性，将会被放在option的data扩展中

方法:
方法名称 参数 描述
setSelected selector, val 设置对应选择器的值
getSelected selector 取得对应选择器的值
getValues none 取得所有联动选择器的值
